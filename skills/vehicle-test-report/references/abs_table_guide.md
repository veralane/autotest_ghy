# ABS测试结果表格结构说明

## 概述

本文档说明ABS测试结果表格的优化结构，支持多车速测试项目和三次重复测试。

## 表格结构设计

### 设计原则

1. **字段作为表头**：将测试指标（平均减速度、制动距离等）作为列标题
2. **要求值作为参考**：在表格底部显示要求值，便于对比
3. **多车速支持**：每个测试项目按车速分组（50km/h、80km/h、100km/h等）
4. **三次重复测试**：每个车速下进行3次测试，并计算平均值

### 表格列定义

| 列名 | 单位 | 说明 |
|------|------|------|
| 测试项目(车速) | km/h | 测试车速，作为行分组标识 |
| 测试序号 | - | 1、2、3或平均 |
| 平均减速度 | m/s² | 制动过程平均减速度 |
| 制动距离 | m | 从制动开始到停止的距离 |
| 减速度相邻峰谷差值 | m/s² | 减速度波动指标 |
| 转向修正角 | deg | 方向盘修正角度 |
| 车轮抱死时间 | s | 单次循环车轮滑移率抱死时间 |
| 附着系数利用率 | % | 路面附着系数利用效率 |
| 主观评分 | 1-10分 | 驾驶员主观评价 |
| 结论 | - | 通过/失败 |

## 数据录入指南

### 1. 填写测试条件

在表格前填写测试环境信息：
- 测试路面类型
- 路面附着系数
- 测试温度和湿度
- 弯道半径（仅弯道测试）

### 2. 录入测试数据

对于每个车速测试项目：
1. 按序号1、2、3分别录入三次测试结果
2. 计算并填写平均值（加粗显示）
3. 填写主观评价

### 3. 判定结论

对比要求值判定：
- 所有指标满足要求 → 通过
- 任一指标不满足要求 → 失败

## 测试项目分类

### 1. 干沥青直线制动
- 测试路面：干沥青
- 测试车速：50、80、100 km/h
- 要求：高附着系数下的制动性能

### 2. 干沥青弯道制动
- 测试路面：干沥青
- 测试车速：50、80 km/h
- 要求：弯道制动稳定性

### 3. 湿沥青直线制动
- 测试路面：湿沥青
- 测试车速：50、80 km/h
- 要求：低附着系数下的制动性能

### 4. 对开路面制动
- 测试路面：对开路面（一侧高附，一侧低附）
- 测试车速：50 km/h
- 要求：不对称路面制动稳定性

## 注意事项

1. **数据一致性**：确保三次测试的测试条件一致
2. **平均值计算**：数值型字段取算术平均值，结论字段根据平均值重新判定
3. **主观评分**：由专业测试驾驶员给出，范围1-10分
4. **结论判定**：需综合考虑客观指标和主观评价
