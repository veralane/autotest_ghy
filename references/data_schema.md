## 字段说明

### 基本信息

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| project_name | string | 是 | 项目名称 |
| report_id | string | 是 | 报告编号 |
| report_date | string | 是 | 报告日期 (YYYY-MM-DD) |

### 前言和概述

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| foreword | string | 否 | 前言内容 |
| overview | string | 否 | 概述内容 |
| test_object | string | 否 | 测试对象描述 |
| test_objectives | array | 否 | 测试目标列表 |
| test_period | string | 否 | 测试周期 |

### 版本信息

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| software_version | string | 否 | 软件版本号 |
| hardware_version | string | 否 | 硬件版本号 |
| firmware_version | string | 否 | 固件版本号 |
| version_change | string | 否 | 版本变更说明 |

### 测试环境

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| test_site | string | 否 | 测试场地 |
| test_equipment | string | 否 | 测试设备 |
| test_tools | string | 否 | 测试工具 |
| environment | string | 否 | 环境条件 |

### 测试结论

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| total_cases | number | 否 | 测试用例总数 |
| passed_cases | number | 否 | 通过用例数 |
| failed_cases | number | 否 | 失败用例数 |
| conclusion | string | 否 | 测试结论 |

### 风险信息

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| risks | array | 否 | 风险列表 |
| risks[].description | string | 是 | 风险描述 |
| risks[].level | string | 是 | 风险等级 (高/中/低) |
| risks[].mitigation | string | 是 | 规避措施 |
| risks[].status | string | 是 | 状态 |

### ABS测试结果

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| abs_straight_braking | object | 否 | 干沥青直线制动测试结果 |
| abs_curve_braking | object | 否 | 干沥青弯道制动测试结果 |

#### ABS测试结果字段定义

每个ABS测试结果对象包含以下字段：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 测试路面 | string | 测试路面类型描述 |
| 测试项目 | string | 测试项目名称 |
| 测试次数 | number | 测试执行次数 |
| 平均减速度(m/s²) | number | 制动过程平均减速度 |
| 制动距离(m) | number | 制动距离 |
| 减速度相邻峰谷差值(m/s²) | number | 减速度波动指标 |
| 转向修正角(deg) | number | 转向修正角度 |
| 单次循环车轮滑移率抱死时间(s) | number | 车轮抱死时间 |
| 路面附着系数 | number | 路面附着系数 |
| 附着系数利用率 | number | 附着系数利用率(%) |
| 主观评分 | number | 主观评分(1-10分) |
| 主观评价 | string | 主观评价描述 |
| 结论 | string | 测试结论(通过/失败) |

### TCS测试结果

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| tcs_results | array | 否 | TCS测试结果列表 |
| results[].case_id | string | 是 | 用例编号 |
| results[].test_item | string | 是 | 测试项目 |
| results[].expected | string | 是 | 预期结果 |
| results[].actual | string | 是 | 实际结果 |
| results[].conclusion | string | 是 | 结论 (通过/失败) |
