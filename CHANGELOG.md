# 更新日志

## 2026-04-08 - ABS测试结果表格优化

### 变更内容

#### 1. 表格结构优化

**优化前：**
- 简单的两列表格（字段 | 测试要求）
- 不支持多车速测试
- 不支持多次重复测试

**优化后：**
- 多列表格，字段作为表头
- 支持不同车速测试项目（50、80、100 km/h）
- 每个车速包含三次测试结果
- 自动计算平均值
- 底部显示要求值便于对比

#### 2. 新增测试项目

- **5.1.1 干沥青直线制动**：50、80、100 km/h
- **5.1.2 干沥青弯道制动**：50、80 km/h
- **5.1.3 湿沥青直线制动**：50、80 km/h
- **5.1.4 对开路面制动**：50 km/h

#### 3. 数据结构更新

更新了 `data_schema.md` 中的ABS测试结果数据结构：
- 新增 `test_conditions` 对象存储测试条件
- 新增 `test_items` 数组支持多车速测试
- 每个测试项包含 `test_runs` 数组（3次测试）
- 新增 `average` 对象存储平均值
- 新增 `requirements` 对象存储要求值

#### 4. 新增文档

创建了 `abs_table_guide.md`，包含：
- 表格结构设计说明
- 数据录入指南
- JSON数据格式映射
- 测试项目分类说明

### 表格字段说明

| 字段 | 单位 | 说明 |
|------|------|------|
| 测试项目(车速) | km/h | 测试车速 |
| 测试序号 | - | 1、2、3或平均 |
| 平均减速度 | m/s² | 制动过程平均减速度 |
| 制动距离 | m | 制动距离 |
| 减速度相邻峰谷差值 | m/s² | 减速度波动指标 |
| 转向修正角 | deg | 转向修正角度 |
| 车轮抱死时间 | s | 车轮抱死时间 |
| 附着系数利用率 | % | 附着系数利用率 |
| 主观评分 | 1-10分 | 主观评价 |
| 结论 | - | 通过/失败 |
